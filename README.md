# TDDでTODOアプリ
## 機能
### 認証
- ユーザー登録できること
- ユーザーログインできること
- ログアウトできること
- セッションで状態を管理する

### タスク
- タスク一覧表示
    - ステータスで絞り込みできるように
        - 未対応
        - 対応中
        - 対応済み
    - 行にステータスを表示し、切り替えられるように
- タスク登録
    - タイトル、詳細、期限が登録可能
        - 必須
            - タイトル
        - 任意
            - 詳細
            - 期限
- タスク更新
- タスク削除
